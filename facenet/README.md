### 基于FaceNet实现人脸识别

### 1. 准备：
| 工具         | 版本     |
|------------|--------|
| python     | 3.9.6  |
| numpy      | 1.24.1 |
| opencv     | 4.7.0  |
| matplotlib | 3.6.3  |

> numpy 安装  
> $ pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple numpy  

> opencv 安装  
> $ pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python  

> matplotlib 安装  
> $ pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple matplotlib
